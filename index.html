ReddApi Javascript Wrapper Usage Examples

Before Starting

By default, these examples will attempt to communicate directly with ReddAPI.com. These attempts will be futile as your browser will block the HTTP Requests due to the same origin policy. To get around this, you need to start chrome with security turned off. like so:

google-chrome --disable-web-security

Do that at your own risk. Alternatively, if you have php running here, use the quick proxy I wrote by uncommenting this line:

MY_APP.ReddApi.setBaseUrl('proxy.php?url=https://api.reddapi.com');

Getting Started
